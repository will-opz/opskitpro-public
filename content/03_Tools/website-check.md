# 网站一键诊断 (Website Check) 指南

## 1. 工具简介
OpsKitPro 网站诊断工具是专为 SRE 和网站管理员设计的“快速全检方案”。它模拟全球扫描节点，对目标域名进行 DNS、HTTP、SSL 及 CDN 的全链路取证。

### 核心功能：
- **健康评分 HUD**：0-100 分自动评分，直观反馈站点可用性。
- **SSL 状态取证**：实时抓取证书有效期、颁发者及 TLS 协议版本。
- **CDN 拓扑识别**：检测网站是否受 Cloudflare, Akamai 或 AWS CloudFront 保护。
- **智能排障建议**：根据探测结果自动生成针对性的修复操作（如 502 错误提示）。

---

## 2. 使用步骤
1.  **输入域名**：在搜索框输入 `example.com`（无需手动输入 https://）。
2.  **启动探测**：点击「Analyze」，系统将按顺序执行联通性测试。
3.  **阅读 HUD 报告**：
    -   **Availability**: 绿灯表示全网可通。
    -   **Latency**: 响应耗时。
4.  **查看安全层**：确认 SSL 证书是否即将过期，协议是否为 TLS 1.3。
5.  **导出数据**：点击底部的 `COPY_AUDIT` 按钮，将 JSON 报文发送给运维团队。

---

## 3. 常见场景
- **站点打不开**：快速确认是域名解析问题还是后端服务崩了。
- **刚换了证书**：实时查看全球生效的证书版本是否正确。
- **配置 CDN 缓存**：确认请求是否被边缘节点正确捕获。
