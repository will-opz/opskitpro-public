# IP 深度取证 (IP Lookup) 指南

## 1. 工具简介
IP Lookup 旨在为网络安全和 DevOps 工程师提供高级 IP 元数据展示。相比于普通的 IP 查询工具，OpsKitPro 提供了更多的 BGP 自治系统 (ASN) 和风险评估维度。

### 核心功能：
- **地理位置锁定**：精准定位 IP 所在的城市、经纬度。
- **ASN & ISP 审计**：显示 IP 所属的机房（如 AS16509 AWS）或宽带服务商。
- **Proxy 检测**：标记 IP 是否来源于已知的代理、VPN 或 Tor 节点。
- **雷达可视化**：基于地理坐标的 HUD 动态雷达扫描图。

---

## 2. 使用步骤
1.  **自动识别**：进入页面，默认显示你当前的出口公网 IP。
2.  **手动查询**：输入任何 IPv4 或域名以开始跨域取证。
3.  **结果解析**：
    -   **Location Card**: 查看国家、城市及经纬度。
    -   **ASN Card**: 识别来源组织（Org）。
4.  **安全审计**：检查 `Security_Audit` 模块，确认该 IP 的风险水位。
5.  **数据交互**：使用 `QUICK_COPY` 按钮快速复制 IP 或完整的地理信息。

---

## 3. 常见场景
- **恶意攻击排查**：发现攻击 IP，通过 ASN 判定是否为受控机房流量。
- **网络优化**：查看目标 IP 的所属运营商，分析跨网延迟原因。
- **用户画像**：验证用户注册地与 IP 库是否匹配。
