# 如何深度查询 IP 信息：从地理位置到 ASN 架构

## 一、问题描述
在排查网络攻击、优化 CDN 路由或验证用户身份时，我们经常需要知道：
- 这个 IP 是哪个国家的？
- 它是家庭宽带还是机房服务器？
- 它是不是一个透明代理或 VPN 节点？

## 二、关键元数据解析
1.  **ASN (Autonomous System Number)**：自治系统编号。通过它你能知道流量是来自阿里云、腾讯云还是中国电信。
2.  **ISP (Internet Service Provider)**：互联网服务提供商。
3.  **Geolocation**：经纬度及城市信息。

## 三、排查技巧

### 1. 识别“机房 IP”
如果一个 IP 的 ASN 归属于 AWS (AS16509) 或 Google (AS15169)，则基本可以判定为服务器流量而非真人。

### 2. 检查 Proxy 标记
代理 IP 经常被用于自动化爬虫或恶意攻击，通过特定库可以检测其 `proxy` 属性。
